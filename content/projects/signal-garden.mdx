## Overview

Signal Garden is a portfolio-ready case study: a data-heavy UI that stays calm and cinematic.

- A clean dashboard layout
- Fast, low-friction controls
- Visuals that feel like an instrument

## Notes

The main design constraint: *make complexity feel like atmosphere*. If the UI looks stressed, the user will be stressed.

> Telemetry is just music with worse marketing.
